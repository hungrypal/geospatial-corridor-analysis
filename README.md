# 🚀 Geospatial Corridor Analysis using Google Earth Engine

## 📌 Overview
This project analyzes infrastructure development along a corridor using satellite data.

## 🛰️ Data Used
- Sentinel-2 (COPERNICUS/S2_SR_HARMONIZED)
- SRTM DEM

## ⚙️ Methodology
- Corridor segmentation (3 km)
- NDVI, BSI, NDBI computation
- Time-series analysis (2024–2025)
- Segment-wise classification:
  - Not Started
  - Under Construction
  - Completed
- Cost surface modeling
- Optimal path extraction

## 📊 Key Outputs
- Vegetation change (NDVI)
- Construction detection (BSI)
- Built-up growth (NDBI)
- Segment classification
- Final optimized corridor

## 🔍 Insights
- Vegetation decreases → land clearing
- BSI increases → construction activity
- NDBI increases → road completion
- Construction progresses sequentially
- Terrain affects construction speed
